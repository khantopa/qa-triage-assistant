  3. Test navigates back to member domain → `isAfterLoginPcrEligible()` in `modalRules.ts` sees `pcr_hide_after_login=true` → AfterLoginPCR modal is suppressed
  4. "Complete my profile" button (rendered by the AfterLoginPCR modal) is never shown → element not enabled → test fails
